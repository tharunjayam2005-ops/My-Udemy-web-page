# Udemy Website Clone

A responsive front-end clone of the Udemy learning platform built using **HTML, CSS, and JavaScript**. This project recreates the core layout and user interface of the Udemy homepage, providing a modern and interactive learning platform experience.

## Features

* Responsive navigation bar with logo, search bar, and action icons
* Category-based menu navigation
* Hero section with course search functionality
* Recommended courses section with course cards, ratings, instructors, and pricing
* Popular topics section with interactive topic tags
* Popular courses showcase
* Fully responsive design for desktop, tablet, and mobile devices
* Smooth hover effects and animations
* Interactive JavaScript functionality for menus, buttons, and course cards
* Modern UI inspired by the official Udemy website

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Font Awesome Icons
